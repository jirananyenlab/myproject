@keyframes

@keyframes เป็นสิ่งเอาไว้กำหนด animation ว่าตอนไหนจะให้ทำอะไร โดย @keyframes จะต้องมีชื่อของมันเอง และชื่อห้ามซ้ำกัน ส่วน keyframes-selector ของ @keyframes มี 2 แบบ คือ 0% - 100% กับ from กับ to (เหมือน 0% กับ 100%)

transform

transform คือ การที่จะใส่เอฟเฟ็กต่าง ๆ ให้แก่ element โดยมีวิธีดังนี้ translate() - การย้าย element ไปตำแหน่งที่กำหนด, rotate() - การหมุน element, scale() - การย่อขยาย element, skewX() - ทำให้เอียงในแนวนอน, skewY() - ทำให้เอียงในแนวตั้ง, matrix() - เป็นการรวม transform ทั้งหมดยกเว้น rotate เข้าด้วยกันในบรรทัดเดียว
!! learn more !!

transition

CSS Transitions เป็น feature ที่ช่วยให้สามารถกำหนดได้เองว่าในการเปลี่ยนแปลงจาก value เก่าไปสู่ value ใหม่ ของในแต่ละ property นั้น จะให้เป็นไปในรูปแบบไหน หรือพูดง่ายๆ ก็คือ เราสามารถกำหนดให้ html element ค่อยๆ เปลี่ยน value ในแต่ละ property ทีละนิด ๆ ได้ ไม่ใช่เริ่มเปลี่ยนจาก 0 แล้วไป 100 เลย เป็นต้น
!! learn more!!

content

content มีไว้แทรกข้อความ หรือสัญลักษณ์ต่างๆ เอาไว้ ด้านหน้า หรือ ด้านหลังกล่องที่เราสั่งมัน content นี้ มันจะทำงานเมื่อใช้คู่กับ :before หรือ :after เสมอ ถ้าสั่งเดี่ยวๆ เลยจะไม่ทำงาน
!! learn more !!

animation

animation คือ shorthand property หรือการเขียนแบบย่อ โดยการใส่ค่าควรเรียงลำดับจาก animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count, animation-direction infinite คือ วนไม่หยุด
!! learn more !!

-webkit-

WebKit คือ render engine ของผลิตภัณฑ์หลายๆตัวใน Mac OS X System Framework เช่น Safari Dashboard Mail และอื่นๆอีกมากมาย ซึ่งต้องขอย้ำว่า WebKit ไม่ใช่ browser แต่เป็น render engine สำหรับ browser และ รองรับ html5 โดยมีผู้พัฒนาหลัก คือ Adobe, Apple, Google สนับสนุนหลายplatformsเช่น Mac OS X Microsoft Windows Symbian iOS Android Qt GTK+ และอีกมากมาย
!! learn more !!

cursor

cursor หรือเมาส์ที่ชี้มักจะแสดงในรูปลูกศร หรือเป็นรูปมือเวลาเลื่อนเมาส์ไปชี้ที่ LINK ต่างๆ แต่ถ้าต้องการตกแต่งเว็บไซต์ให้มีความน่าสนใจยิ่งขึ้น การเปลี่ยนรูป cursor ก็เป็นอีกเทคนิคหนึ่งที่สามารถทำได้ง่าย ๆ
!! learn more!!

object-fit

object-fit สิ่งนี้ใช้บอกให้รูปภาพตอบสนองอย่างไรกับ สิ่งที่มันบรรจุอยู่ เช่น "รักษาอัตราส่วนภาพนั้นไว้!" หรือ "ยืดภาพซะและใช้พื้นที่ให้มากที่สุด!"
!! learn more!!

filter

เป็นการใส่ Effect พื้นฐานให้กับรูปภาพได้ เช่น การทำภาพมัว (Blur), เปลี่ยนภาพให้เป็นสีเทา (Glayscale), การปรับความอิ่มตัวของสี (Saruration) และอื่นๆ
!! learn more!!

text-stroke

คือการกำหนดขอบตัวอักษร แต่ในไม่กี่ปีที่ผ่านมานี้ (Safari, Chrome, iPhone) มีเพียงแค่ webkit เท่านั้นที่ support มัน
!! learn more !!

::before

เป็น css pseudo-elements โดย before และ after อยู่ใน css version 3 ซึ่งประโยชน์ คือ ใช้ใส่ข้อความ สัญลักษณ์ หรือรูปไอคอนต่าง ๆ ไว้ก่อน หรือ หลัง tag HTML ที่เราเลือกไว้ ซึ่งสามารถใส่ได้ทุก tag เมื่อใส่แล้วจะทำให้มีข้อความ สัญลักษณ์ หรือไอคอน ปรากฏอยู่ก่อน-หลัง tag สิ่งที่เราจะใส่ต้องอยู่ใน attribute content:'' ซึ่งในนั้น ค่า Default ของ element นั้น จะเป็น inline element แต่เราสามารถใช้ CSS บังคับ style การแสดงผลให้ แสดงในแบบต่าง ๆ ได้
!! learn more !!

ตกแต่ง scrollbar

ออกแบบ scrollbar ได้ตามต้องต้องการด้วย CSS selector 3 ตัว ได้แก่ ::-webkit-scrollbar เป็นส่วนของ scrollbar ทั้งหมด, ::-webkit-scrollbar-track เป็นส่วนของ progress bar ในตัวของ scrollbar, ::-webkit-scrollbar-thumb เป็นส่วนของ scrollbar ที่ลากขยับได้
!! learn more !!

inset

inset CSS คือคำสั่งแบบสั้นสำหรับกำหนดตำแหน่ง top, right, bottom และ left โดยสามารถกำหนดค่าทั้ง 4 ผ่านคำสั่ง inset เพียงคำสั่งเดียว คล้ายกับการกำหนดค่าของ margin และ padding โดย inset : 5px 8px 9px 15px โดยมีความหมายดังนี้ top ขยับ elements จากด้านบน ขนาด 5px right ขยับ elements จากทางขวา ขนาด 8px bottom ขยับ elements จากด้านล่าง ขนาด 9px left ขยับ elements จากทางซ้าย ขนาด 15px
!! learn more !!

Scroll Snap

scroll-snap-type คือ property ที่ระบุว่าข้อมูลจะเลื่อนไปทางแกนใด x , y มี snapping strictness หรือค่ากำหนดความเข้มงวดมากน้อยของการตรึงวัตถุ และ scroll-snap-align เป็นการกำหนดจุด snap ของ elements โดยเมื่อเลื่อน scroll แล้วจะให้อยู่ในตำแหน่งไหน เช่น ขวา กลาง ซ้าย
!! learn more !!
